# 课程设计


## 学生信息表设计
| 中文名称 | 表名 | 字段属性 | 备注 |
|---------|-----|---------|------|
| 学号 | sno | int(4) | 主码 |
| 姓名 | sname | varchar(10) | 不能为空 |
| 年龄 | sage | smallint | 不能为空 |
| 性别 | ssex | char(4) | 男/女 |
| 专业号 | mno | int(4)| 外码 不能为空 |
| 状态 | state | int(4)| 默认为1且该属性隐藏不被学生看到 |


## 学院信息表设计
| 中文名称 | 表名 | 字段属性 | 备注 |
|---------|-----|---------|------|
| 专业号 | mno | int(4) | 主码 |
| 专业名 | mname | char(32) | 不能为空 |
| 学院名 | mdept | char(32) | 不能为空 |


## 学生成绩表设计
| 中文名称 | 表名 | 字段属性 | 备注 |
|---------|-----|---------|------|
| 学号 | sno | int(4) | 主码 |
| 课程名 | cname | char(64) | 主码 |
| 成绩 | score | int(4) | 0-100 |
